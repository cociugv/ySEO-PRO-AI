# ySEO-PRO-AI Domain

Language for an AI-native SEO system that helps practitioners assess and improve websites.

## Language

**SEO Operator**:
A technical SEO specialist, freelancer, or agency practitioner who uses AI-enabled development tools and manages roughly 5–100 websites.
_Avoid_: End user, webmaster, generic user

**SEO Remediation**:
A proposed website change tied to an SEO finding, with enough evidence to preview, apply, verify, and reverse it safely.
_Avoid_: Suggestion, generated fix, recommendation

**Applied Remediation**:
An SEO remediation that changed its intended target and subsequently passed verification.
_Avoid_: Generated artifact, successful handler, attempted fix

**Rollback**:
Restoration of the target to its state before an applied remediation.
_Avoid_: Undo suggestion, regenerate previous content
