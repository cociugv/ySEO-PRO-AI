# Use external projects only for clean-room research

ySEO-PRO-AI may study external projects to identify user needs, behavioral requirements, workflows, and limitations, but it will not copy their code, prompts, documentation, or assets. Implementations must be written originally from project-owned specifications and standards so ySEO-PRO-AI remains an independently authored product without third-party code-attribution obligations.

## Consequences

Research artifacts must describe behavior without reproducing protected expression. Contributions copied or closely ported from another repository are not accepted, even when that repository uses a permissive license.
